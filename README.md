# [ZHKU Thesis] 仲恺农业工程学院学位论文 Latex 模板

## 🔈 模板简介

一款基于 [ucasthesis](https://github.com/mohuangrui/ucasthesis)，并按照[《仲恺农业工程学院研究生学位论文撰写规范（2023年修订）》](https://yjs.zhku.edu.cn/info/1060/4945.htm)做样式适配的 Latex 模板。
